# voice123-test

App for searching voice actors. Powered by Voice123. [Go to demo!](https://voice123-test.netlify.app/))

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Run your unit tests

```
npm run test:unit
```
